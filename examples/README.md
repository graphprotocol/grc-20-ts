# Examples

Example code demonstrating how to use `@graphprotocol/grc-20`.

## Ranks

- `ranks/create-ordinal-rank.ts` — Create a positional rank (1st, 2nd, 3rd...)
- `ranks/create-weighted-rank.ts` — Create a scored rank with numeric values

## Running

```bash
npx tsx examples/ranks/create-ordinal-rank.ts
```
